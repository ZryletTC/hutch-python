cli.py¶

This module defines the command-line interface arguments for the hutch-python script. It also provides utilities that are only used at startup.

usage: hutch-python [-h] [--cfg CFG] [--debug] [--sim] [--create CREATE]
                    [script]

Launch LCLS Hutch Python

positional arguments:
  script           Run a script instead of running interactively

optional arguments:
  -h, --help       show this help message and exit
  --cfg CFG        Configuration yaml file
  --debug          Start in debug mode
  --sim            Run with simulated DAQ
  --create CREATE  Create a new hutch deployment
setup_cli_env Parse the user’s arguments and gather the session’s objects.
hutch_ipython_embed Make a shell, customize it, then run it
run_script Basic shortcut to running a script in the current hutch python scope.
start_user Picks hutch_ipython_embed or run_script based on the args.
